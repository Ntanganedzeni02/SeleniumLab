Welcome to Selenium Practice App

This application is designed for practicing Selenium WebDriver automation. It includes multiple pages and scenarios to help you master:

  • Locators: XPath, CSS selectors, IDs, classes, and more
  • Waits: Implicit, explicit, and fluent waits with loading elements
  • Page Object Model (POM): Well-structured pages for easy mapping
  • Forms: Text inputs, dropdowns, checkboxes, radio buttons, and file uploads
  • Tables & Pagination: Dynamic data with sorting and pagination
  • Alerts & Popups: JavaScript alerts, iframes, and new window handling
  • Advanced Elements: Hidden elements, tooltips, drag & drop, and more
  • API Integration: Simulated API calls with dynamic content loading
